# Story 1.5: Interrupt Handling

## Status: Done

## Story

**As a** user of claude-auto-resume,  
**I want** to be able to interrupt the tool cleanly with Ctrl+C,  
**so that** I can stop waiting if I change my mind or need to terminate early.

## Acceptance Criteria

1. Capture SIGINT signal (Ctrl+C) during wait periods
2. Display friendly exit message when interrupted
3. Clean up any temporary state or resources
4. Ensure no orphaned processes remain after interruption
5. Handle interruption during different phases (parsing, waiting, executing)
6. Exit with appropriate exit code when interrupted
