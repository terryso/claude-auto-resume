# Story 2.2: Environment Variable Configuration

## Status: Done

## Story

**As a** user of claude-auto-resume,  
**I want** to configure the tool's behavior using environment variables,  
**so that** I can customize timeouts, logging, and safety features without modifying the script.

## Acceptance Criteria

1. Support `CLAUDE_AUTO_RESUME_WAIT_BUFFER` for extra wait time (seconds)
2. Support `CLAUDE_AUTO_RESUME_SKIP_PERMISSIONS` to control permission skipping
3. Support `CLAUDE_AUTO_RESUME_LOG_FILE` for optional log file path
4. Display configured values in help or with `--check` flag
5. Provide sensible defaults when environment variables are not set
6. Validate environment variable values for safety
